Navigation & Mouse Interaction

AutoDrive Editor features a powerful map canvas that allows you to navigate and interact with your road network easily using mouse and keyboard combinations.

Map Navigation

  • Pan (Move View): Click and drag with the Right Mouse Button (RMB).
  • Zoom (Wheel): Use the Mouse Wheel to zoom in and out at the cursor position.
  • Zoom (Vertical): Hold Ctrl + RMB drag (up to zoom in, down to zoom out).
  • Zoom to Window: Use the "Window zoom" tool in the toolbar, then drag with the Left Mouse Button (LMB) to define the area.

Selection

  • Single Selection: Click on a waypoint or route with the Left Mouse Button (LMB).
  • Toggle Selection: Hold Ctrl + LMB click to add/remove an element from the selection.
  • Path Selection: Select a waypoint, then hold Shift + LMB click on another waypoint to select the shortest path between them.
  • Rectangular Selection (Waypoints): Click and drag with the LMB on empty space to select multiple waypoints.
  • Rectangular Selection (Routes): Hold Shift + LMB drag on empty space to select routes.

Keyboard Shortcuts

  • + / -: Zoom in/out.
  • 0: Fit map to window.
  • 1: Reset view to 100%.
