Portable Mode

AutoDrive Editor can be used in "portable" mode, meaning it stores all its settings and configurations in its own folder instead of the system's user folder.

Activating Portable Mode

Portable mode is automatically activated by the presence of a file named settings.xml located in the same folder as the ADEditor.exe executable.

If this file is missing, the application will use the standard system storage (usually in %APPDATA% on Windows).

[!IMPORTANT] It is essential to keep the settings.xml file in the executable's folder. If you delete or move it, the application will switch to normal mode and create new settings on the host computer.

Content of settings.xml

To manually create a portable mode, you can create a text file named settings.xml with the following content:

<?xml version='1.0' encoding='UTF-8'?>
<Settings-file Version='1.0'>
</Settings-file>
