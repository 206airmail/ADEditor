Translating the Application

You can help translate AutoDrive Editor into a new language by following these simple steps.

1. GUI Translation (Interface)

  • Create Folder: Create a new 2-letter folder (the language code, e.g., de, es, it) in the langs/ directory.
  • Prepare PO File: Copy the file langs/adeditor.pot into your new folder and rename it to adeditor.po.
  • Translate: Open this adeditor.po file with a translation software like Poedit.
  • Edit Headers: Fill in the project name, language, and your name in the catalog headers.
  • Save: As you save in Poedit, it will automatically generate a .mo file which the application uses.

2. Documentation Translation (Help)

To provide a complete translation, you should also translate the help files:

  • Create Doc Folder: Create a new folder with the same 2-letter code in the doc/ directory.
  • Copy Base: Copy all files from doc/en/ into your new folder.
  • Translate: Translate the content of the HTML files while keeping the original file names and structure.
  • Update Project: Ensure the .hhc and .hhk files are updated with the translated titles.

The build script Tools/BuildHelpFiles.py will automatically detect your new folder and create the help archive for the application.
