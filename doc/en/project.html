Project Management

AutoDrive Editor uses its own project format (.adproject) to store your road network alongside the map data, ensuring an easy editing experience without modifying your game files until you are ready.

Basic Operations

  • New Project: Creates a fresh workspace. You'll need to specify map names and sizes.
  • Open Project: Loads an existing .adproject file.
  • Save / Save As: Persists your changes to the project file.

Farming Simulator Integration

The core strength of the editor is its seamless integration with Farming Simulator 25 savegames.

  • Import from Savegame: When creating a new project, you can select a FS25 savegame folder to automatically import the map background and the current AutoDrive_config.xml.
  • Export to FS Savegame: Once your edits are complete, use this tool to write your road network back to the game's AutoDrive_config.xml. The editor will offer to create a backup before overwriting.
  • Restore to FS Savegame: Allows you to restore the original AutoDrive_config.xml from your project backup if needed.
