Editing the Network

The road network consists of Waypoints connected by Routes. You can also define Markers as specific destinations.

Waypoints

  • Create: Hold Ctrl + LMB click on empty space to create a new waypoint.
  • Move: Click and drag (LMB) any waypoint to move it. If multiple waypoints are selected, they will all move together.
  • Delete: Select waypoints and press the Delete key.

Routes (Connections)

  • Segment: Select two waypoints and click the "Connect" tool to create a straight segment between them.
  • Curve: Select two waypoints and use the "Curve" tool. A dialog will allow you to define the curve's tension and the number of intermediate points (Bezier curve).
  • Direction: Use the "Direction" tool to swap the direction of a single segment, or "Reverse" to invert the whole selected path.

Markers

  • Add: Select a waypoint and use the "Add Marker" tool to define a destination.
  • Groups: Organize markers into groups (e.g., "Fields", "Points of Interest") to better manage them in the game.
  • Manage: You can edit or delete markers via the toolbar or the context menu.
