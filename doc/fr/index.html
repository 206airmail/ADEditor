Bienvenue dans AutoDrive Editor

AutoDrive Editor est un éditeur de trajets convivial pour le mod AutoDrive de Farming Simulator 25.

Sujets d'Aide Principaux

  • Gestion de Projet : Chargement, sauvegarde et intégration FS25.
  • Navigation et Interactions Souris : Comment se déplacer sur la carte.
  • Édition du Réseau : Création de waypoints, routes et marqueurs.
  • Mode Portable : Utilisation de l'éditeur sans installation.
  • Outils et Raccourcis : Outils de validation et récapitulatif clavier.
  • Traduire l'Application : Comment contribuer à de nouvelles langues.
