Mode Portable

AutoDrive Editor peut être utilisé en mode "portable", ce qui signifie qu'il stocke tous ses réglages et configurations dans son propre dossier plutôt que dans le dossier utilisateur du système.

Activation du Mode Portable

Le mode portable est automatiquement activé par la présence d'un fichier nommé settings.xml situé dans le même dossier que l'exécutable ADEditor.exe.

Si ce fichier est absent, l'application utilisera le stockage standard du système (généralement dans %APPDATA% sur Windows).

[!IMPORTANT] Il est impératif de laisser le fichier settings.xml dans le dossier de l'exécutable. Si vous le supprimez ou le déplacez, l'application basculera en mode normal et créera de nouveaux réglages sur l'ordinateur hôte.

Contenu du fichier settings.xml

Pour créer manuellement un mode portable, vous pouvez créer un fichier texte nommé settings.xml avec le contenu suivant :

<?xml version='1.0' encoding='UTF-8'?>
<Settings-file Version='1.0'>
</Settings-file>
