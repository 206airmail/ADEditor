Gestion de Projet

AutoDrive Editor utilise son propre format de projet (.adproject) pour stocker votre réseau routier en parallèle des données de la carte, garantissant une édition facile sans modifier vos fichiers de jeu avant d'être prêt.

Opérations de Base

  • Nouveau Projet : Crée un nouvel espace de travail. Vous devrez spécifier le nom et la taille de la carte.
  • Ouvrir Projet : Charge un fichier .adproject existant.
  • Enregistrer / Enregistrer Sous : Sauvegarde vos modifications dans le fichier projet.

Intégration Farming Simulator

La force de l'éditeur réside dans son intégration transparente avec les sauvegardes de Farming Simulator 25.

  • Importer depuis une sauvegarde : Lors de la création d'un projet, vous pouvez sélectionner un dossier de sauvegarde FS25 pour importer automatiquement l'image de fond et le fichier AutoDrive_config.xml actuel.
  • Exporter vers une sauvegarde : Une fois vos modifications terminées, utilisez cet outil pour réinjecter votre réseau dans le AutoDrive_config.xml du jeu. L'éditeur proposera de créer une sauvegarde de sécurité avant d'écraser.
  • Restaurer une sauvegarde : Vous permet de restaurer le fichier AutoDrive_config.xml d'origine à partir de la sauvegarde interne du projet si nécessaire.
