Traduire l'Application

Vous pouvez aider à traduire AutoDrive Editor dans une nouvelle langue en suivant ces quelques étapes.

1. Traduction de l'Interface (GUI)

  • Création du dossier : Créez un nouveau dossier de 2 lettres (le code langue, ex: de, es, it) dans le dossier langs/.
  • Préparation du fichier PO : Copiez le fichier langs/adeditor.pot dans ce nouveau dossier et renommez-le en adeditor.po.
  • Traduction : Éditez les en-têtes du fichier et mettez à jour les traductions en utilisant un logiciel comme Poedit.
  • Enregistrement : En enregistrant avec Poedit, un fichier .mo sera automatiquement généré, c'est celui-ci qui est utilisé par l'application.

2. Traduction de la Documentation (Aide)

Pour une traduction complète, vous devriez également traduire les fichiers d'aide :

  • Dossier Doc : Créez un nouveau dossier avec le même code de 2 lettres dans le dossier doc/.
  • Copie de base : Copiez-y tous les fichiers du dossier doc/en/.
  • Traduction : Traduisez le contenu des fichiers HTML en conservant les noms de fichiers et la structure d'origine.
  • Mise à jour projet : Assurez-vous que les fichiers .hhc et .hhk sont mis à jour avec les titres traduits.

Le script Tools/BuildHelpFiles.py détectera automatiquement votre nouveau dossier et créera l'archive d'aide pour l'application.
