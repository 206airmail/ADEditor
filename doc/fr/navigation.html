Navigation et Interactions Souris

AutoDrive Editor propose un canevas de carte puissant qui vous permet de naviguer et d'interagir facilement avec votre réseau routier en utilisant des combinaisons de souris et de clavier.

Navigation sur la Carte

  • Déplacement (Pan) : Cliquez et glissez avec le bouton DROIT de la souris (RMB).
  • Zoom (Molette) : Utilisez la molette de la souris pour zoomer/dézoomer à la position du curseur.
  • Zoom (Vertical) : Maintenez Ctrl + glissement RMB (vers le haut pour zoomer, vers le bas pour dézoomer).
  • Zoom par Fenêtre : Utilisez l'outil "Zoom fenêtre" de la barre d'outils, puis tracez une zone avec le bouton GAUCHE (LMB).

Sélection

  • Sélection Simple : Cliquez sur un waypoint ou une route avec le bouton GAUCHE (LMB).
  • Basculer la Sélection : Maintenez Ctrl + clic LMB pour ajouter/retirer un élément de la sélection.
  • Sélection de Chemin : Sélectionnez un waypoint, puis maintenez Shift + clic LMB sur un autre waypoint pour sélectionner le chemin le plus court entre les deux.
  • Sélection Rectangulaire (Waypoints) : Cliquez et glissez avec le LMB sur un espace vide pour sélectionner plusieurs waypoints.
  • Sélection Rectangulaire (Routes) : Maintenez Shift + glissement LMB sur un espace vide pour sélectionner les routes.

Raccourcis Clavier de Vue

  • + / - : Zoom avant/arrière.
  • 0 : Ajuster la carte à la fenêtre.
  • 1 : Réinitialiser la vue à 100%.
